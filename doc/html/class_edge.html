Detailed Description

template<class Traits>
class Edge< Traits >

Esta classe representa segmentos de linha orientados. As Edge podem ter 2 ou mais nós, um nó em cada extremo e o restante no seu interior com espaçamento constante. Uma Edge com N nós tem a seguinte forma:

*n0_____*n2______*n3____ ... ____*nN-1_____*n1

Uma Edge de N+1 nós é dita Edge de ordem N, e sua orientação é definida como a direção ao longo da Edge de n0 até n1.
