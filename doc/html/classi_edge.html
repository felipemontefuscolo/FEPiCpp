Detailed Description

template<class Traits>
class iEdge< Traits >

Esta classe representa segmentos de linha orientados. As iEdge podem ter 2 ou mais nós, um nó em cada extremo e o restante no seu interior com espaçamento constante. Uma iEdge com N nós tem a seguinte forma:

*n0_____*n2______*n3____ ... ____*nN-1_____*n1

Uma iEdge de N+1 nós é dita iEdge de ordem N, e sua orientação é definida como a direção ao longo da iEdge de n0 até n1.
