Detailed Description

template<class Traits>
class iPolygon< Traits >

A classe iPolygon representa os polígonos (triângulo, quadrângulos, ...), entidades da malha de dimensão 2. Dependendo da ordem, os polígonos dessa classe podem ter arestas curvas.

Note:
Esta é uma classe de conceito abstrato (apesar se não ser abstrata sob o ponto de vista da linguagem c++ pois não tem funções virtuais puras), logo não se deve instanciá-la a não ser que se saiba exatamente o que se está fazendo.
