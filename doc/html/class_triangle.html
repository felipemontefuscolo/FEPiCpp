Detailed Description

template<class Traits>
class Triangle< Traits >

A classe de triângulos. A dimensão de seu tipo na malha é 2, mas não se deve confundir com a dimensão do espaço onde o triângulo está, que pode ser 2 ou 3. O triângulo tem no mínimo 3 nós (um em cada vértice; triângulo linear). Um triângulo de ordem n tem (n+1)(n+2)/2 nós.
