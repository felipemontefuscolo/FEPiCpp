Detailed Description

template<class Traits>
class iHalfEdge< Traits >

Armazena o iD da célula incidente e o seu índice local em relação a esta célula.

O iD da iHalfEdge é feita da seguinte composição

<f, ith>

onde

  • f:= iD da célula (polígono) [ 28bits ]
  • ith := i-ésima aresta desta célula [ 4bits ]

Essas informações são armazenadas em uma única variável (uint) [32bits] usando manipulação de bits. Os valores de f e ith estão, respectivamente, na faixa:

  • 0 <= f <= 268 435 456
  • -1 <= ith <= 14
Warning:
o tipo unsigned int deve ter 32 bits.
