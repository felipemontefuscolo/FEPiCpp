Detailed Description

Interface para armazenar um rótulo e 3 flags. Esses atributos são compactados em um único char. Os 3 primeiros bits do char são para armazenar os flags:

  • char[0] := "isdead" ?;
  • char[1] := "must be killed" ?;
  • char[2] := "other flags"

O restante é para armazenar o rótulo (inteiro de 0 a 31 se o char for de 8 bits).
