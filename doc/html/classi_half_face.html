Detailed Description

template<class Traits>
class iHalfFace< Traits >

Armazena o iD da célula incidente, o seu índice local em relação a esta célula e a sua âncora (ver definição no final da descrição).

O iD da iHalfEdge é feita da seguinte composição

<v, ith, anchor>

onde

  • v := iD da célula (poliedro) [ 27bits ]
  • ith := i-ésima face desse volume [ 3bits ]
  • anchor := k-ésimo nó da face do volume [ 2bits ]

Essas informações são armazenadas em uma única variável (uint) [32bits] usando manipulação de bits. Os limites de v, ith e anchor são, respectivamente:

  • 0 <= v <= 134 217 727
  • -1 <= ith <= 6
  • 0 <= anchor <= 4

definição para âncora : dada uma iHalfFace que contenha os vértices n0,n1,...,nB, se ela é a face F e âncora A de um poliedro, então, o nó B-A é o primeiro nó da face F do poliedro.
